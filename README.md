# ⚽ Álbum Copa 2026

App mobile (Flet) para contabilizar figurinhas do álbum Panini Copa do Mundo 2026.

## Como baixar o APK

1. Vá na aba **Actions** deste repositório
2. Clique no build mais recente (✅ verde)
3. Role até **Artifacts** e baixe `album-copa-2026-apk`
4. Descompacte o `.zip` → instale o `.apk` no celular

## Recursos

- 980 figurinhas (códigos reais: BRA1, GER5, FWC3...)
- Filtro por setor, seleção, status e código
- Aba de repetidas para trocar
- Cores por grupo de seleções
